遍历所有驱动器，对每个驱动器创建一个线程并行加密 -> 生成AES-128密钥key -> 遍历磁盘文件，用key对其进行AES加密 -> 从程序中获取一个公钥pubkey（内置固定） -> 用pubkey加密key，得到ekey -> 将ekey写入README.TXT -> 释放key
