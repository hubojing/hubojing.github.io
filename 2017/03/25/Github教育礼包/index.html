Github教育礼包

2017-03-25
  • Github

感谢小伙伴让我知道github教育礼包，并教我怎么申请：

  1. 到教育优惠页面申请：https://education.github.com/
  2. 填写学校信息及申请理由，如果有edu邮箱一般很快就通过了；如果没有，需要提供证件来证明。
  3. 一般来说一小时到几天就申请成功了。（我当天晚上收到了邮件）
    有用的主要三个东西：
  4. me域名免费一年
  5. DO的免费VPS代金券50美金（有效期1年），可以使用最低配置11个月
  6. 免费SSL证书
