目标

java编写一个控制台版的“达达租车系统”

功能

1.展示所有可租车辆
2.选择车型、租车量
3.展示租车清单，包含：总金额、总载货量及其车型、总载人量及其车型

思路

　　首先定义一个Car类，它包含基本功能：车名、载客数、载货量、日租金。接着创建三个小类，分别是客车类、货车类和皮卡类（既能载客又能载货），它们都继承Car类。最后需要一个主类，用于开启整个系统，调用每个小类。
