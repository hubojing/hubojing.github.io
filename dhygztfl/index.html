Hexo根据更新时间修改文章排序机制

Apr 12, 2024
阅读时长: 1 分钟

　　改一行代码就行了。

前言（可不看）

之前由于电脑不在身边，长期通过其它电脑写文章利用之前说过的github action推送博客更新。但是发现一个问题，它总是按照文章创建时间排序，而不是文章更新时间。 查过网上很多方法，法一：博客源目录配置文件新增updated_option: mtime，不行。 法二：利用插件， 新增

解决方法

hexo源目录下，\node_modules\hexo-generator-index\lib\generator.js 将

'use strict';

var pagination = require('hexo-pagination');

module.exports = function(locals){
  var config = this.config;
  var posts = locals.posts.sort('-updated'); //修改这里！ 原代码为var posts = locals.posts.sort('-date');
  var paginationDir = config.pagination_dir || 'page';

  return pagination('', posts, {
    perPage: config.index_generator.per_page,
    layout: ['index', 'archive'],
    format: paginationDir + '/%d/',
    data: {
      __index: true
    }
  });
};

就这么简单。
